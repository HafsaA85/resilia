# Noor for Her 🌿

**Noor for Her** is a gentle, faith-friendly web application designed to support women in understanding anxiety, building emotional clarity, and developing quiet confidence — one step at a time.

This project provides a safe and private space for reflection, journaling, and self-awareness without pressure or perfection.

---

## ✨ Features

- 🧠 **Anxiety Triggers Tracker**  
  Log situations, thoughts, and intensity levels to identify emotional patterns.

- ✍️ **Personal Journal**  
  Write private reflections linked to specific triggers or as free journaling entries.

- 🔥 **Reflection Streak**  
  Visual streak showing recent journaling consistency.

- 🌱 **Insights Dashboard**  
  View average anxiety levels, recent activity, and common triggers.

- 🌙 **Calm & Faith-Friendly Design**  
  Created to feel reassuring, gentle, and emotionally safe.

- 🔒 **Private & Secure**  
  All entries are user-specific and protected by authentication.

---

## 🛠 Tech Stack

- **Backend:** Django (Python)
- **Frontend:** HTML, CSS (Bootstrap)
- **Database:** SQLite (development)
- **Authentication:** Django Auth
- **Version Control:** Git & GitHub
